## Competencias Desarrolladas

- Conversión entre sistemas numéricos: decimal, binario y hexadecimal
- Comprensión del modelo OSI y sus 7 capas
- Identificación de protocolos seguros (HTTPS, SSH, SFTP) e inseguros (Telnet, HTTP, FTP)
- Diseño de topologías de red con **Cisco Packet Tracer** (switch, servidor, DHCP)
- Configuración de IPs estáticas y servidor DHCP en red local
- Gestión de usuarios, grupos y permisos en Linux (`useradd`, `chmod`, `chown`)
- Configuración de firewall con **firewalld** (puertos TCP 80, 443, 20, 21)
- Reconocimiento de red con **Nmap** (host discovery, port scanning, detección de versiones)
- Identificación de CVEs con Nmap (`CVE-2017-5638`)
- Escaneo de vulnerabilidades web con **Nikto**
- Escaneo de vulnerabilidades con **OpenVAS/Greenbone** sobre Metasploitable 2
- Reconocimiento de direccionamiento IPv4 e IPv6 (unicast global, link-local)
- Captura y análisis de tráfico de red con **Wireshark**
- Conocimiento de normativas: PCI DSS y Ley 21.459 (Chile)
