# Lab 02 - Administración Linux: Usuarios, Permisos y Firewall

**Asignatura:** Ciberseguridad en Sistemas Operativos y Redes  
**Fecha:** 15-10-2025

---

## Objetivos

- Crear y administrar usuarios y grupos en Linux
- Gestionar permisos de archivos y directorios (chmod, chown)
- Identificar procesos que consumen más recursos del sistema
- Configurar reglas de firewall con firewalld
- Capturar y analizar tráfico de red con Wireshark

---

## Temas Cubiertos

- Gestión de usuarios en Linux (`useradd`, `passwd`)
- Creación de grupos (`groupadd`)
- Creación de directorios y archivos (`mkdir`, `touch`)
- Permisos Linux: `rwx r-x r--` (chmod 754)
- Propietario y grupo dueño (`chown`)
- Monitoreo de procesos (`top`, `ps`)
- Configuración de firewalld (puertos TCP 80, 443, 20, 21)
- Captura de tráfico HTTP con Wireshark (`frame contains`)
