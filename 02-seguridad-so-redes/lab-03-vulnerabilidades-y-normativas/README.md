# Lab 03 - Escaneo de Vulnerabilidades, Análisis de Red y Normativas

**Asignatura:** Ciberseguridad en Sistemas Operativos y Redes  
**Fecha:** 19-11-2025  
**Autor:** Nicolás Zamora

---

## Objetivos

- Generar un reporte de vulnerabilidades con OpenVAS sobre una máquina virtual
- Identificar la vulnerabilidad CVE-2017-5638 con Nmap
- Escanear puertos y versiones de servicios en Metasploitable con Nmap
- Capturar y analizar tráfico HTTP con Wireshark
- Ejecutar análisis web con Nikto sobre Metasploitable
- Describir la triada CIA de seguridad
- Explicar la normativa PCI DSS
- Describir la Ley 21.459 de Chile

---

## Temas Cubiertos

- OpenVAS/Greenbone: escaneo y reporte de vulnerabilidades
- Nmap: identificación de CVE y escaneo de servicios (`-sV`)
- Wireshark: captura y análisis de tráfico HTTP
- Nikto: análisis de vulnerabilidades web
- Triada CIA (Confidencialidad, Integridad, Disponibilidad)
- Normativa PCI DSS
- Ley 21.459 (Delitos Informáticos en Chile)
